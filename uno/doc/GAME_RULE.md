# UNO 二人规则

本实现面向 Bot 对战裁判，使用稳定、可复现的二人 UNO 规则。

## 基本配置

- 固定 2 名玩家，座位为 `0` 和 `1`。
- 使用标准 108 张 UNO 牌：
  - 红、黄、绿、蓝各 25 张。
  - 每色数字 `0` 一张，`1` 到 `9` 各两张。
  - 每色 `Skip`、`Reverse`、`Draw Two` 各两张。
  - `Wild` 四张，`Wild Draw Four` 四张。
- 每名玩家起手 7 张。
- 初始弃牌堆顶固定抽到一张有颜色的数字牌，避免开局动作牌效果歧义。
- 玩家 `0` 先行动。

## 合法出牌

一张牌满足任一条件即可打出：

- 颜色等于当前颜色。
- 数字等于弃牌堆顶数字。
- 动作类型等于弃牌堆顶动作类型，例如 `Skip` 接 `Skip`。
- `Wild` 和 `Wild Draw Four` 总是可出。

打出 `Wild` 或 `Wild Draw Four` 时，动作必须同时声明下一轮颜色：

```text
play:<card_id>:red
play:<card_id>:yellow
play:<card_id>:green
play:<card_id>:blue
```

## 动作牌效果

- `Skip`：对手跳过，当前玩家继续行动。
- `Reverse`：二人局中等同于 `Skip`，当前玩家继续行动。
- `Draw Two`：对手摸 2 张并跳过，当前玩家继续行动。
- `Wild`：声明当前颜色，然后轮到对手。
- `Wild Draw Four`：声明当前颜色，对手摸 4 张并跳过，当前玩家继续行动。

本实现不包含叠加摸牌、质疑 `Wild Draw Four`、喊 UNO 或惩罚摸牌。

## 摸牌和跳过

- 只在没有任何可出牌时，合法动作才包含 `draw`。
- `draw` 摸 1 张并结束回合，不允许立即打出刚摸到的牌。
- 当摸牌堆为空时，会保留弃牌堆顶牌，其余弃牌洗回摸牌堆。
- 如果没有可出牌且无法摸牌，合法动作是 `pass`。
- 双方连续 `pass` 时，游戏以 `blocked` 结束，手牌更少者获胜，手牌数相同则平局。

## 胜负

- 玩家打出最后一张手牌时立即获胜，状态为 `empty_hand`。
- 达到 `max_plies` 时以 `turn_limit` 截断，手牌更少者获胜，手牌数相同则平局。
- Bot 抛异常、超时或返回非法动作时，裁判判该 Bot 负。
