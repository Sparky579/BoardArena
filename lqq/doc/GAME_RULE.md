# 路墙棋规则

本文档描述本项目实现的路墙棋规则范围。该版本为双人对战，目标是从己方起点走到对面底线。

## 基本设置

- 支持玩家数：固定 `2` 人。
- 棋盘大小：`9 x 9`。
- 玩家 `0` 初始位置：`[8, 4]`，目标是到达第 `0` 行任一格。
- 玩家 `1` 初始位置：`[0, 4]`，目标是到达第 `8` 行任一格。
- 每名玩家初始拥有 `10` 堵墙。
- 玩家 `0` 先手。

## 回合行动

每回合行动玩家必须选择一种合法动作：

- 移动：向上、下、左、右移动一格；如果该方向紧邻对手且对手后方或侧边可通行，则可以跳到对手后方或侧边一格。
- 放墙：放置一堵横墙或竖墙。

## 移动规则

- 通常只能移动到棋盘内相邻格。
- 不能穿过墙。
- 不能移动到对手当前所在格。
- 如果目标方向的相邻格被对手占据，并且对手后方一格在棋盘内且没有墙阻挡，则可以沿该方向跳过对手。
- 如果目标方向的相邻格被对手占据，也可以斜向跳到对手侧边一格；例如己方在对手下方时，可以跳到对手左边或右边。目标格必须在棋盘内，且对手到目标格之间不能有墙阻挡。

## 放墙规则

- 墙长为两格，方向为横墙 `H` 或竖墙 `V`。
- 放墙坐标为墙的左上原点 `(row, col)`，范围均为 `0..7`。
- 横墙 `H,row,col` 阻挡 `[row, col]` 到 `[row + 1, col]`，以及 `[row, col + 1]` 到 `[row + 1, col + 1]`。
- 竖墙 `V,row,col` 阻挡 `[row, col]` 到 `[row, col + 1]`，以及 `[row + 1, col]` 到 `[row + 1, col + 1]`。
- 墙不能重叠。
- 墙不能在同一个原点交叉。
- 放墙后必须保证双方都仍然至少存在一条通往目标底线的路径。

## 胜负

- 玩家 `0` 到达第 `0` 行即获胜。
- 玩家 `1` 到达第 `8` 行即获胜。
- 如果达到裁判回合上限仍未分出胜负，状态为 `turn_limit`，胜者为 `None`。
- 如果极端局面下当前玩家没有任何合法动作，状态为 `no_legal_actions`，对手获胜。

## 坐标说明

- 坐标使用 `[row, col]`。
- `row` 从上到下递增，范围 `0..8`。
- `col` 从左到右递增，范围 `0..8`。
