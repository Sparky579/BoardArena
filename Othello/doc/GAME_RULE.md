# 黑白棋规则

本文档为 `BoardArena/Othello` 的规则介绍。环境实现标准 8x8 黑白棋/Othello 规则。

## 基本设置

- 支持玩家数：固定 `2` 人。
- 玩家 `0`：黑方，先手。
- 玩家 `1`：白方，后手。
- 初始棋子：黑方在 `e4`、`d5`，白方在 `d4`、`e5`。
- 棋盘坐标：文件为 `a..h`，横线为 `1..8`。

## 行动格式

所有动作使用字符串：

- 普通落子：棋盘坐标，例如 `d3`、`c4`。
- 强制跳过：`PASS`。

玩家有合法落子时不能选择 `PASS`。只有当前玩家没有合法落子、且对手仍有合法落子时，`PASS` 才会出现在 `legal_actions` 中。

## 合法落子

一次落子必须至少在一个方向上夹住连续的对方棋子。方向包括横、竖、斜八个方向。被新落子和己方已有棋子夹住的对方棋子全部翻成己方棋子。

## 终局

满足任一条件时结束：

- 棋盘已满。
- 黑白双方都没有合法落子。
- Bot 对战中达到环境 `max_plies` 上限时，以 `turn_limit` 截断。

终局时棋子更多的一方获胜；数量相同则平局。

## 异常结束

Bot 对战中：

- Bot 抛异常：状态为 `bot_exception`，该 Bot 判负。
- Bot 返回非法动作：状态为 `invalid_action`，该 Bot 判负。
- Bot 单步决策超时：状态为 `timeout`，该 Bot 判负。

常见 `status`：

- `disc_majority`
- `draw`
- `turn_limit`
- `invalid_action`
- `bot_exception`
- `timeout`
