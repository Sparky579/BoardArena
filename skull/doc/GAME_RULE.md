# Skull 简化规则

## 基本设置

- 支持 `2..6` 名玩家。
- 每名玩家初始有 `3` 张花 `F` 和 `1` 张骷髅 `S`。
- 每名玩家有手牌、暗置牌堆和分数。
- 先到 `2` 分，或其他玩家都没有牌时获胜。

## 放牌阶段

- 当前玩家可以暗放一张手牌：`PLAY_F` 或 `PLAY_S`。
- 仍有牌的玩家都至少放过一张牌后，才允许开始叫号。
- 叫号动作为 `BID_n`，其中 `n` 不能超过桌面暗牌总数。

## 叫号阶段

- 进入叫号阶段后，玩家可以 `PASS`，或用更大的 `BID_n` 加叫。
- 除最高叫号者外，其他未出局玩家都 `PASS` 后，最高叫号者进入挑战。

## 挑战阶段

- 挑战者先自动翻开自己牌堆顶部的牌，直到翻完自己的牌或达到叫号数量。
- 如果还需要继续翻牌，挑战者选择 `FLIP_i` 翻开玩家 `i` 的牌堆顶部一张牌。
- 翻到任意 `S`，挑战失败。
- 全部翻到 `F`，挑战成功并得到 `1` 分。

## 失败惩罚

- 挑战失败者会从自己的全部剩余牌中随机损失一张。
- 损失后所有人的暗置牌堆清空，剩余牌回到手牌。
- 如果某名玩家没有任何剩余牌，该玩家出局；只剩一名有效玩家时，该玩家获胜。
